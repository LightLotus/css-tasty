CSS Introduction

CSS is the language we use to style an HTML document. CSS describes how HTML elements should be displayed.

What is CSS?

  • CSS stands for Cascading Style Sheet.
  • CSS describes how an HTML should look like in a web page and how it can control the layout of multiple pages at once.

Three ways to insert CSS in yout HTML document

  • External CSS
  • Internal CSS
  • Inline CSS

External CSS

External style is a file defined within the <link> element and is saved with a ".css" file extension.

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8" />
<meta name="viewport" content="width=device-width, initial-scale=1" />
<title>Basic HTML File</title>
</head>
<body>
<h1>Hello, world!</h1>
</body>
</html>
